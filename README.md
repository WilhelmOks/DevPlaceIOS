# DevPlace iOS

A native iOS app for DevPlace (https://devplace.net) made with SwiftUI.

## DevPlace Swift SDK

A multiplatform Swift SDK for the DevPlace API.

This will later be a separate project in a separte repository. But for now it lives inside of the iOS app project for convenience and easier iteration during the early development phase.

## Status of development

Done:
* sign in, sign out
* forgot password and sign up via redirect to browser
* listing posts in the feed and loading more
* viewing user profiles and own profile
* viewing images and videos attached to posts
* viewing and voting on polls
* upvoting and downvoting posts
